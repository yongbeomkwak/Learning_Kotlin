-   피보나치 수
    -    tailrec(꼬리 호출 제거 재귀함수 키워드)
    -    즉, 공재귀도 스택 공간을 사용하긴 한다.
    -   이 문제를 완전히 없애는 방법은 "공재귀 함수를 루프로 바꾸는 것"이다.
    -   tailrec 키워드를 사용하면 코틀린이 자동으로 꼬리재귀 함수를 루프를 사용한 코드로 변경하여 컴파일한다.

            tailrec fun 함수명(parms..):Type  ...

-   2개 이하로 다른 비트

    - a and b : and 연산
    - a or c :or 연산

    - 2의보수: 반전 후 +1
